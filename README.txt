Orientações para execução do Chat:
1 - Baixe as classes Server.java e Client.java em seu computador.
2 - Abra o prompt de comando. Caso esteja utilizando o sistema operacional Windows,
execute o prompt em modo administrador.
3 - Através do terminal, vá até a pasta onde as classes estão salvas.
4 - Na pasta, compile a classe Server.java através do comando "javac Server.java" (sem as aspas).
5 - Você verá que um arquivo Server com extensao .CLASS foi criado dentro da pasta.
6 - Compile a classe Client.java através do comando "javac Client.java" (sem as aspas).
7 - Você verá que um arquivo Client com extensao .CLASS foi criado dentro da pasta.
8 - Ainda no prompt, execute a classe Server através do comando "java Server" (sem as aspas).
9 - Uma janela de diálogo se abrirá, informando a porta utilizada pelo Server (porta 10001). Pressione "Ok".
10 - Uma outra janela se abrirá, confirmando a execução do servidor. Pressione "Ok".
11 - Abra um novo prompt de comando. Caso esteja utilizando o sistema operacional Windows, execute-o também em modo administrador.
11 - Repita o processo até chegar na pasta onde estão as classes salvas.
12 - Na pasta, execute a classe Client, através do comando "java Client" (sem as aspas).
13 - Uma janela, informando o IP e a porta utilizada, será exibida. NÃO ALTERE ESSAS INFORMAÇÕES!
14 - Nessa janela, haverá um espaço para você informar um nome de usuário. Digite a informação e dê "Ok".
15 - O chat já estará em funcionamento, com a janela de mensagens sendo aberta. Todas as mensagens digitadas, 
serão enviadas ao servidor, porém, não serão entregues a ninguém, pois só há uma conexão.
16 - Para abrir novas janelas e inserir novos usuários no chat, volte ao passo 11, abrindo um novo prompt, executando a classe Client.
17 - ATENÇÃO! As o programa limita as mensagens à um total de 140 caracteres. Caso você digite uma mensagem maior,
o programa informará que o limite de caracteres foi ultrapassado e sua mensagem não será enviada.
18 - Para sair, clique em Sair na tela de usuário, ou digite "Sair" e tecle Enter. Os demais usuários serão notificados que 
você desconectou-se.
